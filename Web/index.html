HTML Text Elements

Element Description End Tag
<h1> – <h6> Headings from largest to smallest </h1>
<p> Paragraph of text </p>
<div> Block-level container for grouping elements </div>
<span> Inline container for styling specific text </span>
<pre> Preformatted text (preserves spaces and line breaks) </pre>
<blockquote> A section quoted from another source </blockquote>
<cite> Title of a creative work </cite>
<q> Short inline quotation </q>
<address> Contact information for the document author </address>
